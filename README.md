# Masonry-layout

## Component Requirement

[test/coding/frontend](https://wiredcraft.gitbook.io/recruitment-test/coding/frontend)

## Tech Stack

- React@17
- Styled Component@5
- Webpack@5
- Typescript@4
- Lerna
- Koa

## Dev

```
npm install
lerna bootstrap
lerna run dev
```

Client will run on port `8080`, and backend api run on `8081` default.

## Test

```
lerna run test
```
