# How to run    
```
git clone https://github.com/EcutDavid/frontend-test.git    
cd frontend-test   
git checkout David  
```  

Local debug:  
```
npm run dev
```
Run test:  
```
npm run test
```
Build prd files:  
```
npm run build
```
